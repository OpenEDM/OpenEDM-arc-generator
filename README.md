## Main Specifications

<div align="center">

| Parameter          | V1                                           | V2                    |
| ------------------ | -------------------------------------------- | --------------------- |
| Input voltage (DC) | max. 150V                                    | 48V                   |
| Input power        | max. 200W                                    | max. 300W             |
| Arc current (peak) | adjustable, max. 30A                         | adjustable, max. 30A  |
| Ignition voltage   | fixed (same as the input voltage), max. 150V | adjustable, max. 175V |

</div>
